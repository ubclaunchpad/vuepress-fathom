# vuepress-fathom ![npm (scoped)](https://img.shields.io/npm/v/@ubclaunchpad/vuepress-fathom)

A simple VuePress plugin for Fathom Analytics.

```sh
npm i -s @ubclaunchpad/vuepress-fathom
```

```js
module.exports = {
  // ...
  plugins: [
    ['@ubclaunchpad/vuepress-fathom', {
      'siteID': 'XXXXXXXX',
    }],
  ],
}
```
